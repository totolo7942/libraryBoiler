# Jmh Java Microbenchmark Harness
>  IntelliJ Plugin Jmh의 경우 JDK1.8 에서만 제공된다.

## JDK11+ 버전에서 처리 방법
* src/jmh 디렉토리 생성
* 해당 로직추가 
* gradle jmh로 실행
